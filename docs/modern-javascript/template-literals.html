Template literals

Template literals là một tính năng mới của ES6 cho phép bạn làm việc với chuỗi một cách mới hơn, dễ hơn, tuyệt vời hơn.

Cú pháp nhìn qua là bạn sẽ thấy nó đơn giản, chỉ cần sử dụng cặp backticks thay vì nháy đơn hay nháy kép.

const a_string = `something`

Nó độc đáo ở chỗ nó cung cấp nhiều tính năng mà các chuỗi bình thường sử dụng nháy đơn hoặc nháy kép không làm được.

  • Cung cấp một cú pháp tuyệt vời để viết chuỗi trên nhiều hàng.

  • Viết biến trong chuỗi chưa bao giờ dễ dàng đến vậy.

  • Cho phép bạn tạo DSL với thẻ mẫu (template tags) (DSL là viết tắt của Domain Specific Language và một ví dụ là nó được sử dụng trong React by Styled Components, để định nghĩa CSS cho một component). Bạn sẽ làm việc với DSL trong các bài học sau về React.

Chúng ta cũng đi sâu vào chi tiết:

Xuống hàng

Trước ES6, để có thể xuống hàng bạn phải dùng ký tự /\.

const string =
  'first part \
second part'

Nó cho phép bạn viết code trên 2 dòng, nhưng khi render thì lại hiển thị chỉ có một dòng. Để render nhiều dòng bạn rõ ràng là cần viết thêm \n tại cuối mỗi dòng:

const string =
  'first line\n \
second line'

hoặc

const string = 'first line\n' + 'second line'

Bây giờ hãy xem cách mà Template literals làm việc.

Khi bạn tạo một chuỗi với cặp dấu backstick ``, bạn chỉ cần nhấn Enter xuống hàng mà không cần thêm bất kỳ ký tự phức tạp nào vào cuối dòng, và nó sẽ render theo ý bạn.

const string = `Hey
this
string
is awesome!`

Hãy nhớ rằng các khoảng trắng là có ý , nên nếu bạn viết

const string = `First
                Second`

sẽ khởi tạo một chuỗi thế này:

First
                Second

Một cách để khắc phục vấn đề này là hãy để trắng dòng đầu tiên và thêm phương thức trim() vào cuối chuỗi sẽ loại bỏ bất kỳ khoảng trắng nào phía trước chuỗi:

const string = `
First
Second`.trim()

Biến trong chuỗi

Template literals cung cấp cách viết biến và biểu thức trong chuỗi cực kỳ tuyệt vời với cú pháp ${...}:

const myVariable = 'test'
const string = `something ${myVariable}` //something test

bên trong ${...} bạn có thể thêm bất kỳ thứ gì, thậm chí là biểu thức:

const string = `something ${1 + 2 + 3}`
const string2 = `something ${foo() ? 'x' : 'y'}`

Rest and spread

Bạn có thể mở rộng một mảng, một object hoặc một string bằng cách sử dụng toán tử spread.

Hãy bắt đầu ví dụ với một mảng

const a = [1, 2, 3]

//Bạn có thể tạo một mảng mới bằng cách viết

const b = [...a, 4, 5, 6]

//Vậy b sẽ là [1, 2, 3, 4, 5, 6]

Dễ hiểu phải không nào, copy một mảng:

const c = [...a]

Điều này cũng hoạt động với object

//Copy một object
const newObj = { ...oldObj }

Với string, toán tử spread cho phép tạo một mảng với mỗi phần tử là một ký tự

const hey = 'hey'
const arrayized = [...hey] // ['h', 'e', 'y']

Toán tử này có một số ứng dụng khá hữu ích. Quan trọng hơn hết là bạn có có khả năng sử dụng một mảng làm đối số cho hàm một cách rất đơn giản.

const f = (foo, bar) => {}
const a = [1, 2]
f(...a)

(trong quá khứ bạn có thể sử dụng bằng cách f.apply(null, a)) nhưng thật xấu xí và khó hiểu phải không.

Rest thì hữu ích khi làm việc với array destructing:

const number = [1, 2, 3, 4, 5]
[first, second, ...others] = number

và spread

const numbers = [1, 2, 3, 4, 5]
const sum = (a, b, c, d, e) => a + b + c + d + e
const sumOfNumber = sum(...numbers)

ES2018 cũng giới thiệu rest properties tạm hiểu là thuộc tính rest, điều này là tương tự nhưng áp dụng cho object.

Rest properties

const { first, second, ...others } = {
  first: 1,
  second: 2,
  third: 3,
  fourth: 4,
  fifth: 5
}
first // 1
second // 2
others // { third: 3, fourth: 4, fifth: 5 }

Spread properties cho phép tạo ra một object mới bằng cách kết hợp các thuộc tính của object được truyền vào sau toán tử spread.

const items = { first, second, ...others }
items //{ first: 1, second: 2, third: 3, fourth: 4, fifth: 5 }
